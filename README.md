# Employee Management System

A responsive Employee Management System built with React and Tailwind CSS. Supports separate dashboards for Admin and Employees with task management functionality.

## 🚀 Live Demo
[View Live](https://employee-management-system-brown-five.vercel.app)

## 🛠️ Tech Stack
- **React** — UI library
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **LocalStorage** — Data persistence

## ✨ Features
- Admin and Employee login roles
- Admin dashboard to manage employees and assign tasks
- Employee dashboard to view and update assigned tasks
- Task status tracking (Active, Completed, Accepted, Failed)
- Data persisted via localStorage
- Fully responsive dark-themed UI
